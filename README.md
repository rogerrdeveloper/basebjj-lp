🥋 BASE Brazilian Jiu-Jitsu Landing Page

A premium landing page developed for BASE Brazilian Jiu-Jitsu, focused on delivering a modern, cinematic, and elegant visual experience.

🚀 Overview

This project is a front-end website built using HTML, CSS, and JavaScript, designed to represent the identity, discipline, and strength of a Brazilian Jiu-Jitsu academy.

The interface follows a clean and premium aesthetic, using light textures and gold accents to create a strong visual presence.

🎯 Purpose
Present the academy professionally
Attract new students
Reinforce brand identity
Simulate a real-world client project
🛠️ Technologies Used
HTML5
CSS3
JavaScript (Vanilla)
🎨 Design Concept

The design is based on a cinematic premium style, combining:

Light textured backgrounds
Gold highlight elements
Minimalist and clean layout
Strong typography
